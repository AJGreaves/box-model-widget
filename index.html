The Box Model

Every HTML element can be represented as a rectangular box. That box has a content area, paddings, margins, and borders. The sum of all of the four areas horizontally and vertically is the total width and height of the element, respectively.

The box model is a core principle of CSS, and it is impossible to code layout without a deep understanding of the box model.

The content in the box model is the content of the element: text, images etcâ€¦ It contains the actual content.

The padding creates space around the content between the content and its border.

The border defines the a border around the element, whenever we want to draw a line around an element, like the edge of a button, we would use the border part of the box model.

The margin creates space around the element itself, between its border and other elements around it.
